## 🎯 프로젝트 소개

**오성역사톡!!**은 중학교 역사 수업을 위해 설계된 올인원 교육용 웹앱입니다.

비상교육 역사 ①(세계사) · 역사 ②(한국사) 교과서를 기반으로, AI 챗봇·AI 퀴즈·인터랙티브 타임라인·교과서 원문 뷰어·게이미피케이션을 하나의 HTML 파일에 담았습니다.

### 왜 만들었나요?

| 문제 | 해결 |
|------|------|
| 역사가 암기 과목이라는 편견 | AI 챗봇 + 게이미피케이션으로 자기주도 학습 유도 |
| 학생별 성취도 파악이 어려움 | 교사 대시보드 + CSV 명단 관리 |
| 다양한 앱 설치의 번거로움 | 단일 HTML 파일, URL 접속만으로 즉시 사용 |
| 저사양 스마트폰 호환 문제 | 경량 설계, 외부 라이브러리 최소화 |

### 벤치마킹

- **Duolingo** — 연속 학습일(Streak), 뱃지, XP 보상 시스템
- **Quizlet** — 마이크로 러닝, 단원별 퀴즈
- **콴다(QANDA)** — AI 기반 즉각적 문제 해결
- **Google Classroom** — 교사 관리 대시보드

## 🚀 시작하기

### 필요 조건

- 웹 브라우저 (Chrome, Safari, Edge, Firefox 등)
- (선택) Google AI Studio API 키 — AI 챗봇·AI 퀴즈 고급 기능용

### 1분 빠른 시작

1. `index.html` 파일을 다운로드합니다.
2. 브라우저에서 파일을 엽니다.
3. 학생 또는 교사로 로그인합니다.
4. 끝! 바로 학습을 시작하세요.

> 💡 AI 기능 없이도 교과서 학습, 타임라인, 기본 퀴즈, 게이미피케이션은 모두 작동합니다.

## 🌐 배포 방법 (GitHub Pages)

### Step 1: GitHub 레포지토리 생성

```
레포지토리 이름: OseongHistoryTalk (또는 원하는 이름)
공개 여부: Public
```

### Step 2: 파일 업로드

레포지토리에 `index.html` 파일을 업로드합니다.

> GitHub 웹에서: `Add file` → `Upload files` → `index.html` 드래그 → `Commit changes`

### Step 3: GitHub Pages 활성화

```
Settings → Pages → Source: "Deploy from a branch"
Branch: main / root → Save
```

### Step 4: 접속 URL 확인

약 1~2분 후 아래 주소로 접속 가능:

```
https://{GitHub사용자명}.github.io/{레포지토리명}/
```

예시:
```
https://kkbhong71.github.io/OseongHistoryTalk/
```

> 이 URL을 학생들에게 공유하면 끝! 앱 설치 없이 브라우저에서 바로 사용할 수 있습니다.

## ⚙️ 설정 가이드

### 🔑 Gemini API 키 설정

AI 챗봇과 AI 퀴즈의 고급 기능을 사용하려면 Google AI Studio의 API 키가 필요합니다.

**API 키 발급 방법:**

1. [Google AI Studio](https://aistudio.google.com/) 접속
2. Google 계정으로 로그인
3. 좌측 메뉴 `Get API key` → `Create API key` 클릭
4. 생성된 키 복사 (형식: `AIzaSy...`)

**코드에 적용:**

`index.html` 파일을 텍스트 편집기로 열고, 293번째 줄 부근에서:

```javascript
// 변경 전
const GK='YOUR_GEMINI_API_KEY_HERE';

// 변경 후 (예시)
const GK='?';
```

> ⚠️ API 키는 코드에 직접 포함되므로, 레포지토리를 **Public**으로 운영할 경우 키 노출에 유의하세요. 필요 시 Private 레포지토리 사용을 권장합니다.

> 💡 API 키 없이도 기본 키워드 기반 챗봇 응답과 내장 퀴즈 뱅크가 작동합니다.

### 📖 교과서 PDF 연결

현재 비상교육 역사①·② 교과서 PDF가 Google Drive를 통해 연결되어 있습니다. 다른 교과서로 변경하려면:

```javascript
// index.html 내 C 객체에서 did 값 변경
history1: { title:"역사① 세계사", did:"Google_Drive_파일_ID", ... }
history2: { title:"역사② 한국사", did:"Google_Drive_파일_ID", ... }
```

Google Drive 파일 ID는 공유 링크에서 추출합니다:
```
https://drive.google.com/file/d/[여기가_파일ID]/view
```

> ⚠️ Google Drive 파일의 공유 설정을 "링크가 있는 모든 사용자에게 공개"로 설정해야 합니다.

## 📘 사용자 가이드

### 🧑‍🎓 학생 로그인

1. 첫 화면에서 **🧑‍🎓 학생** 탭 선택
2. **반** 선택 (1학년 1반 ~ 3학년 2반)
3. **번호** 입력
4. **이름** 입력
5. **비밀번호** 입력
6. `시작하기 🚀` 클릭

| 항목 | 설명 |
|------|------|
| 초기 비밀번호 | 번호를 4자리로 맞춘 것 (예: 3번 → `0003`, 15번 → `0015`) |
| 비밀번호 변경 | 로그인 후 **설정 > 🔑 비밀번호 변경** |
| 비밀번호 분실 | 교사에게 초기화 요청 |

### 👩‍🏫 교사 로그인

1. 첫 화면에서 **👩‍🏫 교사** 탭 선택
2. 비밀번호 입력: `oseong2026`
3. `교사 로그인` 클릭
4. 자동으로 교사 대시보드로 이동

### 📋 CSV 학생 명단 업로드

교사 대시보드에서 CSV 파일로 학생을 일괄 등록할 수 있습니다.

**CSV 파일 형식:**

```csv
반,번호,이름
1-1,1,김서준
1-1,2,이하은
1-1,3,박도윤
1-2,1,최수아
2-1,1,강지유
3-2,15,홍길동
```

| 규칙 | 설명 |
|------|------|
| 첫 줄 | 헤더 (자동 무시됨) |
| 반 형식 | `학년-반` (예: `1-1`, `2-2`, `3-1`) |
| 번호 | 숫자 (예: `1`, `15`) |
| 이름 | 한글 이름 |
| 인코딩 | UTF-8 (Excel에서 저장 시 "CSV UTF-8" 선택) |

> 💡 교사 대시보드의 `📥 샘플 CSV 다운로드` 버튼으로 양식 파일을 받을 수 있습니다.

**업로드 절차:**

1. 교사 대시보드 > `📂 CSV 파일 선택` 클릭
2. CSV 파일 선택
3. 자동으로 명단 등록 완료
4. 반별 학생 카드가 화면에 표시됨

> 기존 명단에 추가 업로드 시, 같은 반+번호의 학생은 자동 갱신됩니다.

## 🎨 커스터마이징

### 교과서 내용 수정

`index.html` 내 `C` 객체에서 단원·주제 데이터를 수정할 수 있습니다:

```javascript
const C = {
  history1: {
    units: [
      {
        id: "h1-1",
        num: "Ⅰ",
        emoji: "🏛️",
        title: "단원 제목",
        era: "시대 범위",
        topics: [
          {
            title: "주제 제목",
            kw: "유튜브 검색 키워드",
            summary: "한 줄 요약",
            content: "핵심 정리 내용 (▶·• 기호로 구조화)"
          }
        ]
      }
    ]
  }
};
```

### 타임라인 이벤트 추가

`TL` 배열에 새 이벤트를 추가합니다:

```javascript
{ y: "1945", t: "광복", d: "해방", tp: "k", pd: "n" }
// tp: "k"=한국사, "w"=세계사
// pd: "a"=고대, "m"=중세, "n"=근현대
```

### 뱃지 추가

`BD` 배열에 새 뱃지를 추가하고, `cb()` 함수에 획득 조건을 추가합니다.

### 디자인 색상 변경

CSS 변수를 수정하여 전체 테마를 변경할 수 있습니다:

```css
:root {
  --a1: #6C5CE7;  /* 메인 퍼플 */
  --a2: #00CECE;  /* 시안 */
  --a3: #FD79A8;  /* 핑크 */
  --a4: #FDCB6E;  /* 골드 */
  --a5: #00B894;  /* 민트 */
  --a6: #E17055;  /* 코랄 */
}
```

## 💾 데이터 저장 구조

모든 데이터는 브라우저의 `localStorage`에 저장됩니다.

| 키 | 용도 | 비고 |
|----|------|------|
| `oh2` | 학생 개인 학습 데이터 (XP, 뱃지, 활동 등) | 개인별 |
| `oh2_roster` | 교사가 업로드한 학생 명단 JSON | 교사용 |
| `oh2_pw_{반}_{번호}` | 학생 개별 비밀번호 | 예: `oh2_pw_1-1_3` |

> ⚠️ localStorage는 브라우저별·기기별로 독립입니다. 다른 기기에서 접속하면 데이터가 공유되지 않습니다. 서버 기반 동기화가 필요하면 백엔드(예: Firebase, Supabase) 연동이 필요합니다.

## ❓ FAQ

### Q: AI 기능 없이도 사용 가능한가요?
**A:** 네! API 키 없이도 교과서 학습, 타임라인, 내장 퀴즈(5문제), 게이미피케이션이 모두 작동합니다. AI 챗봇은 주요 키워드(고조선, 삼국, 프랑스 혁명 등)에 대한 기본 응답을 제공합니다.

### Q: 학생이 다른 기기에서 로그인하면 데이터가 유지되나요?
**A:** 현재 버전은 localStorage를 사용하므로 기기별로 데이터가 독립적입니다. 서버 기반 동기화가 필요하면 Firebase 등을 연동하는 확장이 가능합니다.

### Q: CSV 파일을 Excel에서 만들 수 있나요?
**A:** 네! Excel에서 작성 후 `다른 이름으로 저장` → `CSV UTF-8 (쉼표로 분리)(*.csv)` 형식으로 저장하면 됩니다.

### Q: 교과서 PDF가 안 보여요.
**A:** Google Drive 파일의 공유 설정이 "링크가 있는 모든 사용자에게 공개"로 되어 있어야 합니다. 학교 Google Workspace 정책에 따라 제한될 수 있습니다.

### Q: 모바일에서도 잘 작동하나요?
**A:** 네! 모바일 퍼스트로 설계되어 스마트폰에서 최적화된 UI를 제공합니다. PWA 지원으로 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.

### Q: Gemini API 무료 사용량은 얼마나 되나요?
**A:** Google AI Studio의 Gemini API는 무료 티어에서 분당 15회, 일일 1,500회 요청이 가능합니다 (2025년 기준). 일반적인 학급 규모에서는 충분합니다.
